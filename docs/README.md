# ChatForge Documentation

Welcome to the official documentation for **ChatForge**, the ultimate AI Terminal Interface. ChatForge combines a powerful, feature-rich chat environment with the sleek, hacker aesthetic of a modern terminal application.

## Overview

ChatForge is designed to give power users a highly modular and customizable interface for interacting with various AI models. It natively supports OpenRouter, allowing you to fluidly swap between AI providers, inject custom system prompts, and seamlessly search your chat histories.

### Key Sections:

1. **[Core Features](./features.md)**: Learn how to master the ChatForge terminal, manage sessions, and interact with individual message blocks.
2. **[Customization & Settings](./customization.md)**: Discover how to personalize the Terminal's look, feel, and AI behaviors through the dynamic Settings Panel.

## Quick Start
1. Run `npm install` and `npm run dev` in the `frontend` directory.
2. Provide your OpenRouter API key on the initial boot screen.
3. Access the terminal by typing <kbd>Enter</kbd> to authenticate. 
4. Type `//>help` in the chat to see all available commands.
